Developer Image Instructions
============================

To add your developer image:

1. Place your professional photo in this folder and name it "developer.jpg"
   - Recommended size: 400x400 pixels or larger (square format works best)
   - Supported formats: JPG, PNG, WebP
   - File name should be: developer.jpg

2. The image will automatically appear on:
   - Homepage (hero section - "Meet Your Developer" card)
   - Contact page (contact details section)

3. If the image is not found, a placeholder will be shown automatically.

4. Image styling:
   - The image is automatically styled with rounded corners
   - It's responsive and will scale on mobile devices
   - Maximum size: 280px on desktop, 200px on tablet, 180px on mobile

Tips for best results:
- Use a professional headshot or portrait
- Ensure good lighting and clear background
- Square or portrait orientation works best
- Keep file size under 500KB for faster loading
